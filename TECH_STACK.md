# Global Truth: Tech Stack

## Architecture & Frameworks
- **Framework:** Flutter
- **Language:** Dart 3
- **State Management:** Riverpod (`flutter_riverpod`)
- **Backend as a Service:** Firebase (Auth, Firestore)
- **Payment Gateway:** Razorpay (Planned)
- **QR Scanner:** `mobile_scanner` (Planned)
- **Image Caching:** `cached_network_image`

## Database
- NoSQL Firestore Database
- Strict Role-Base Access Control (RBAC) via Firebase Security Rules

## Current Setup Status
- Firebase Auth & Firestore fully integrated.
- Google Sign-In implemented and tested for Web/Mobile.
- Real-time data streaming for events and tickets functional.
- Zero analysis issues in the current codebase.
